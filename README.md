# Meeting Archive Repository

This repository serves as an archive for recorded meetings in the Edge Foundation team.

## How to Archive a Meeting

1. Go to the [Issues](../../issues) page
2. Click "New Issue"
3. Select the "Archive Recorded Meeting" template
4. Fill in the required information:
   - **Meeting**: The name of the meeting or subject covered
   - **Date**: In ISO format (YYYY-MM-DD)
   - **URL**: The URL of the recorded meeting
5. Submit the issue

The meeting will be archived automatically in the `meetings/` directory.

## Archived Meetings

All archived meetings can be found in the [`meetings/`](./meetings) directory.
